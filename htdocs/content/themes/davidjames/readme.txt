== Description ==
2015 Theme.

* Responsive Layout
* Stickynav
* Woocommerce
* Contact Form 7

== Installation ==

1. Install Wordpress 4.1 or later
2. Copy davidjames theme to themes folder
3. Install woocommerce and contact form 7
4. Customize and enjoy

== Woo Commerce ==

The woocommerce folder inside the davidjames theme are overrides to the plugin theme files. This allows you to update woocommmerce without overriding your themes changes.
